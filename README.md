# Gym Management System (PulseGym)

A full-stack gym management application that allows gym owners to manage members, plans, and payments efficiently with a secure admin panel.

## Features
- JWT-based secure admin authentication  
- Member management (Add, Update, Delete)  
- Subscription plan management  
- Automated monthly billing system  
- Revenue and membership reporting  
- Payment status tracking (Paid / Pending / Expired)  

---

## Tech Stack
**Backend:**
- Java
- Spring Boot
- Spring Security (JWT)
- JPA & Hibernate
- MySQL

**Frontend:**
- React.js
- Tailwind CSS
